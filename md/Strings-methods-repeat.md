### ![ico25] repeat()

Возвращает новую сроку, содержащую заданное аргументом число копий исходной строки

#### ![cap-25] 1

```javascript
var str = "ХОРОШО! "
console.log ( str.repeat ( 4 ) )
```

в консоли будет:

```
ХОРОШО! ХОРОШО! ХОРОШО! ХОРОШО!
```
