### Сложение строк

Две строки можно просто сложить

Например, в результате выполнения следующего кода:
```javascript
var  str_1 = "Привет, ",   str_2 = "студент"
console.log ( str_1 + str_2 )
```
в консоли будет:
```
Привет, студент
```
