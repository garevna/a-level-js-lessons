### :memo: join ()

Объединяет все элементы массива в строку ( "string" )

Аргумент - символ-разделитель

:coffee: ❶

```javascript
var drinks = [ "кофе", "чай", "сок", "вода", "молоко" ]
var drinkStr = drinks.join ( " | " )
```

Значение **drinkStr**:
```
"кофе | чай | сок | вода | молоко"
```
