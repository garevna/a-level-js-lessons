### :ledger: charCodeAt ()

Возвращает код символа, находящегося в заданной аргументом позиции строки

:coffee: :one:

```javascript
var str = "ХОРОШО"
console.log ( str.charCodeAt ( 4 ) )   //  1064
```

:coffee: :two:

```javascript
var str = "1234567"
console.log ( str.charCodeAt ( 4 ) )  //  53
```
